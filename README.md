## 🕹️ 私のすきなこと：ゲーム分析

ゲームをやり続けているうちに疑問に思ったことがあります。**「なんでゲームってこんなに面白いの？」**。

具体的には「なぜこのアクションを気持ちいいと思うのか」「ついつい続けてしまうのはなぜなのか」といった観点から、ゲームを分析するようになってきました。最近では『バイオハザードre4』をクリア済みです。

### 💡 14年ゲームをやってたからわかるゲームの面白さの本質

- **フィードバックのループ**: 行動に対する即時の「報酬」と「評価」。
- **没入感（イマージョン）**: ストーリー、グラフィック、サウンドの調和。
- **UX/UI の心理学**: プレイヤーの直感に訴えるインターフェース。
- **進行のモチベーション**: 「もっと続けたい」と思わせる未達成の課題と進行状況の可視化。
