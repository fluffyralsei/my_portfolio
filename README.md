###　私の好きな事
私は昔から飽き性で、どんなことをやっても1年は続かないということは珍しくありませんでした。\
小学生のころテニスをやりたい！と思い、親に猛烈に説得してテニススクールに通わせてもらいましたが、5か月で飽きがきてやめてしまいました。\
<br>
そんな自分でも何年も継続していることがあります。それが「ゲーム」です。ゲームとの出会いは幼稚園(当時6歳)までにさかのぼり、親がdsでマリオカートをやっているのを見たのが出会いです。そこでやらせてもらったところゲームに完全にドハマりしてしまいました。\
<br>
あれから約14年経ちましたが未だにゲームに対する熱意は変わりません。\
ゲームをやり続けているうちに疑問に思ったことがあります。「なんでゲームってこんなに面白いの？」。何事もすぐ飽きてしまう私がなぜ14年間も没頭し続けてしまうのか。
そう思い始め「ゲーム設計」という観点でゲームを分析・プレイするようになってきました。\
具体的にはなぜこのアクションを気持ちいいと思うのか、ついつい続けてしまうのはなぜなのかそういった観点からゲームを分析するようになってきました。
